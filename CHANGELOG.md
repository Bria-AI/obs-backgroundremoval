# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-22

### Features

- Real-time background removal powered by Bria V-RMBG 3.0
- High-quality alpha matte with natural edges for hair and fine details
- Integrates as a native OBS Effect Filter — applies to any camera source
- One-time OAuth browser sign-in to authenticate with your Bria account
- Windows (x64), macOS (arm64 + x86_64), and Linux (x86_64) support
- Usage telemetry via PostHog to help improve the plugin
