## [1.0.0] - 2026-06-21

### Features

- Real-time background removal via Bria V-RMBG 3.0 streaming endpoint
- Non-binary alpha edges for natural hair and fine-detail cutouts
- Native OBS Effect Filter integration — works on any camera source
- OAuth browser-based sign-in to Bria (one-time setup)
- Windows (x64) and macOS (arm64 + x86_64) support
- Analytics support
