# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately to **security@bria.ai**.

- **Do not** open a public GitHub issue, pull request, or discussion for
  security vulnerabilities.
- Include enough detail to reproduce the issue (affected version, steps,
  and impact) where possible.
- We will acknowledge your report and follow up with next steps.

Thank you for helping keep Bria Background Removal for OBS and its users safe.
